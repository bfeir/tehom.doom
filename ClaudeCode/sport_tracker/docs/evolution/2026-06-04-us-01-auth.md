## Business Context

This delivery completed the auth test suite for the React PWA. Previously only scenario 1 (returning user bypass) was active. The 6 newly activated scenarios provide contract-level coverage of the full authentication flow:

- **Scenario 2**: Email sign-up — new user can create an account
- **Scenario 3**: RLS isolation — user A cannot read user B's data
- **Scenario 4**: Offline first-use message — descriptive plain-language error when offline before auth
- **Scenario 5**: Invalid credentials — plain-language guidance (not a raw Supabase error)
- **Scenario 6**: Network failure no-stack-trace — error message is human-readable, no internal details leaked
- **Scenario 7**: JWT expiry queue preservation — offline queue is preserved and re-queued after token refresh

## Key Decisions

**mapAuthError exported (was module-private)**
Acceptance-test-level assertion required importing the function to validate exact error message contracts. Exporting it from AuthScreen.tsx was the minimal change needed. The function is a pure deterministic mapping with no side effects, so exposure carries no risk.

**Scenario 5 uses anon client instead of service-role client**
The service-role client bypasses Supabase Auth entirely and does not enforce password validation. Using it in scenario 5 would have produced a false pass. Switching to the anon client is the correct test of the actual user-facing authentication path.

**Scenarios 6+7 replaced expect(true).toBe(true) with real assertions**
The original placeholder assertions (tautological) gave no contract coverage. Scenarios 6 and 7 were replaced with: mapAuthError assertions verifying human-readable output, and OfflineQueue/authStore structural assertions verifying queue preservation behavior.

**Delivery-scoped mutation kill rate: 85.7%**
Stryker was scoped to the mapAuthError function. The overall Stryker score of 41% is an artifact of mutating the full AuthScreen.tsx file including UI rendering code outside this delivery's scope. The delivery-scoped rate is the meaningful signal.
